# Imágenes de Autenticación

Esta carpeta contiene las ilustraciones utilizadas en las páginas de autenticación.

## Imágenes requeridas:

### 1. `login-illustration.svg` o `.png`
- **Tamaño recomendado**: 512x512px o similar
- **Formato**: SVG preferido (escalable) o PNG con fondo transparente
- **Uso**: Página de inicio de sesión
- **Tema**: Relacionado con trabajo, tareas, productividad o colaboración

### 2. `reset-password-illustration.svg` o `.png`
- **Tamaño recomendado**: 512x512px o similar
- **Formato**: SVG preferido (escalable) o PNG con fondo transparente
- **Uso**: Página de recuperación de contraseña
- **Tema**: Seguridad, protección, recuperación de acceso

## Sugerencias de diseño:

- **Estilo**: Moderno, minimalista, ilustración plana
- **Colores**: Que combine con el gradiente azul-morado del fondo
- **Colores sugeridos**: Blancos, azules claros, morados claros, o tonos pasteles
- **Evitar**: Colores muy oscuros que no contrasten con el fondo

## Recursos de ilustraciones gratuitas:

- [unDraw](https://undraw.co/) - Ilustraciones SVG personalizables
- [Storyset](https://storyset.com/) - Ilustraciones animadas y estáticas
- [DrawKit](https://www.drawkit.com/) - Paquetes de ilustraciones gratis
- [Blush](https://blush.design/) - Ilustraciones de diferentes artistas
- [Illustrations](https://illlustrations.co/) - Open source illustrations

## Nota técnica:

El componente tiene un fallback automático al emoji si la imagen no se encuentra,
por lo que la aplicación seguirá funcionando incluso sin las imágenes.
